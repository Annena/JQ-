统计说明

全部菜品统计：含单菜、套餐（套餐所属菜品）

均价：销售金额/销售数量

销售总量: 销售数量+退菜数量

销售总额: 销售金额 + 退菜金额

退菜数量: 退的菜品数量

退菜金额: 退的菜品金额

销售数量：菜品的售卖数量（含赠送数量，不含退菜数量）

销售金额：所有订单菜品的原价合计（含增菜金额，不含退菜金额）

赠送数量：赠送的菜品数量

净销售量: 销售数量 - 赠送数量

净销售额: 销售金额 - 赠送金额

折扣金额：会员价优惠 + 会员折扣优惠 + 银台折扣 + 套餐优惠

结账金额: 净销售额 - 折扣金额

优惠金额: 赠送金额 + 折扣金额 + 支付方式优惠部分

实收金额: 结账金额 - 支付方式优惠部分

销售量占比：分类销量占比 = 分类销售数量/总销售数量；单个菜品销量占比 = 单个菜品销售数量/所属分类销售数量

结账金额占比：分类结账金额占比 = 分类结账金额/总结账金额；单个菜品结账金额占比 = 单个菜品结账金额/所属分类结账金额

折扣率：折扣金额/销售金额
